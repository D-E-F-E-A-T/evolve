## Requirements
* Nim and any Unix OS with Any of:
* Yay
* Pacman
* Paludis
* Portage & Layman

## Installation Instruction
```bash
$ git clone https://github.com/Uniminin/evolve
$ cd evolve
$ make
$ make install (needs superuser acess, i.e do 'sudo make install') 
```
